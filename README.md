# 🎮 AI Tic-Tac-Toe Game

A modern Tic-Tac-Toe game built using HTML, CSS, and JavaScript with an unbeatable AI opponent powered by the Minimax algorithm.

---

## 🚀 Live Demo
🔗 https://bittu169.github.io/Tic-Tac-Toe_Game/

---

## ✨ Features

- 🤖 AI Opponent (Minimax Algorithm)
- 🎯 Turn Indicator (Player vs AI)
- 🎉 Popup Winner Display
- 🟩 Winning Box Highlight
- 📊 Scoreboard (Player vs AI)
- 🔊 Sound Effects
- 📱 Responsive Design
- 🔄 Reset & New Game Options

---

## 🧠 How AI Works

The AI uses the **Minimax Algorithm**, which:
- Evaluates all possible moves
- Chooses the best optimal move
- Ensures the AI never loses
